# 👋 CONTRIBUTING — как прислать свою серию

Мы открыты к гостевым выпускам.

## Принимается

- Рассказы 4–12 страниц A4/A5
- Сценарии в Markdown или txt
- Листы персонажей в нашем стиле
- Переводы на другие языки

## Не принимается

- Контент с участием несовершеннолетних в любом эротическом/сексуальном контексте
- Пропаганда наркотиков, нацистская/фашистская символика, призывы к насилию
- Откровенное порно (лёгкая эротика допустима в трагикомическом ключе)
- Использование реальных имён/фото без согласия

## Куда

- Issue с тегом `guest-episode`
- Pull Request в ветку `guests/`
- email: через профиль @metapeka

## Чек-лист для PR

- [ ] Сценарий в Markdown или txt
- [ ] Листы персонажей в `assets/characters/`
- [ ] Соответствие канону из SUPPORTING.md
- [ ] Лицензия CC BY-NC-SA 4.0
- [ ] Указаны источники референсов
